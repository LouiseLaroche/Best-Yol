# Bestyol backend

## Techs

-   Express with Typescript

## Requirements

-   [pnpm installed](https://pnpm.io/installation)

    if you have node and npm already installed:

    `npm install -g pnpm`

-   [node.js installed](https://nodejs.org/en/download/package-manager)

## Installation

```bash
pnpm i
```

## Usage

dev env:

```bash
pnpm run dev
```

build:

```bash
pnpm build
```

prod:

```bash
pnpm start
```
